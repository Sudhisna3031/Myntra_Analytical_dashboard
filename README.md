# 🛍️ Myntra Sales Analysis Dashboard (Power BI)

## 📊 Project Overview

This project presents a comprehensive **Sales Analysis Dashboard** for Myntra using **Power BI**. It provides insights into sales performance, customer preferences, product distribution, and regional trends.

The dashboard helps stakeholders make **data-driven decisions** by visualizing key metrics and trends effectively.

---

## 🚀 Key Features

* 📌 **Total Sales Analysis** – Overall revenue performance
* 📦 **Order Insights** – Total number of orders placed
* 💸 **Discount Tracking** – Average discount applied
* 🛒 **Product Count** – Number of unique products
* ⭐ **Customer Ratings** – Average rating analysis

---

## 📈 Dashboard Insights

### 🔹 Sales & Discount Trends

* Tracks monthly sales performance
* Compares total sales with average discount percentage

### 🔹 Category-wise Sales

* Sales distribution across **Men & Women categories**
* Identifies high-performing segments

### 🔹 Top Brands Performance

* Highlights top 3 brands:

  * H&M
  * Roadster
  * Puma

### 🔹 Brand-wise Sales

* Revenue comparison across brands like:

  * Roadster
  * WROGN
  * HERE&NOW

### 🔹 Order Distribution

* Gender-based order split:

  * Women: 761
  * Men: 423

### 🔹 Rating Analysis

* Shows distribution of product ratings
* Helps identify customer satisfaction trends

### 🔹 Product Distribution

* Category-wise product availability (t-shirts, shirts, jeans, etc.)

### 🔹 Geographic Insights

* Orders mapped across different states in India

## 🧰 Tools & Technologies Used
